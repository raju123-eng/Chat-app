# 💬 Real-Time Chat Application

A simple and responsive real-time chat application built using **Node.js**, **Express.js**, and **Socket.IO**.  
This project allows multiple users to connect and exchange messages instantly through a web browser.

---

# 🚀 Features

✅ Real-time messaging  
✅ Username-based chat  
✅ Online users list  
✅ WhatsApp-style UI  
✅ Message timestamps  
✅ Responsive frontend design  

---

# 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML | Structure of webpage |
| CSS | Styling and UI design |
| JavaScript | Frontend interaction |
| Node.js | Backend runtime |
| Express.js | Server framework |
| Socket.IO | Real-time communication |
